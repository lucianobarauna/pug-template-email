# pug-template-email
Template de email generate with pug

The idea is to have a json and consummate to facilitate the maintenance of the code and development

## Install
Using npm
```
npm i
```

Using yarn
```
yarn
```

## Start development
Using npm
```
npm start
```

Using yarn
```
yarn start
```

## Build
Using npm
```
npm run build
```

Using yarn
```
yarn build
```
